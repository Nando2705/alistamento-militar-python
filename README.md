# Alistamento Militar em Python

Projeto desenvolvido durante meus estudos de Python.

## Objetivo

Verificar a situação do alistamento militar com base no ano de nascimento e sexo informado pelo usuário.

## Conceitos praticados

* Entrada de dados
* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas condicionais aninhadas
* Manipulação de datas com `datetime`
* Operadores matemáticos
* Formatação de strings (`f-strings`)

## Como executar

1. Instale o Python.
2. Execute o arquivo:

```bash
python alistamento.py
```

## Exemplo

Ano de nascimento: 2008

Qual é o seu sexo (M/F): M

Ainda faltam 2 ano(s) para o alistamento.

Seu alistamento será em 2028.
